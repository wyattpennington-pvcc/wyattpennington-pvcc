Program Terms

  • Machine Language: A low-level programming language consisting of binary code that a computer's central processing unit (CPU) can directly execute.
  • Debugging: The process of finding and fixing errors or bugs in a computer program.
  • Assembler: A program that translates assembly language into machine code.
  • Assembly Language: A low-level programming language that is a human-readable representation of machine language.
  • Translator: A program that converts source code from one programming language to another.
  • Compiler: A type of translator that translates the entire source code of a program into machine code before execution.
  • Interpreter: A type of translator that translates and executes source code line by line.
  • Cross Platform System: A system or software that can run on different types of computer platforms.
  • Natural Language: A human language used for communication, such as English or Spanish.
  • Formal Language: A language with a specific set of rules and syntax, often used in programming.
  • Machine Independence: The ability of a program to run on different types of machines without modification.
  • High-Level Programming Language: A programming language that is closer to human language and abstracts away low-level details.
  • Object-Oriented Programming: A programming paradigm that uses objects and classes for organizing code.
  • Scripting Language: A lightweight programming language often used for automation and scripting tasks.
  • Data Type: A classification of data that determines the type of operations that can be performed on it.
    • Float
    • Integer
    • String
  • Primitive Data Type: A basic data type that is not composed of other data types.
  • Data Structure: A way of organizing and storing data to perform operations efficiently.
  • Python Collection Data Structures:
    • List
    • Tuple
    • Set
    • Dictionary
  • Function: A reusable block of code that performs a specific task.
  • Local Variable: A variable that is declared within a specific block of code and is only accessible within that block.
  • Global Variable: A variable that is declared outside of any specific block of code and is accessible throughout the program.
  • Source Program: The original program written by the programmer.
  • Object Program: The machine code generated by a compiler or assembler from the source program.
